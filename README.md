# ngFilesort

[![npm package][npm-ver-link]][releases]
[![][dl-badge]][npm-pkg-link]

Sort angular files by their dependencies.

### Install

```a
npm install ng-filesort
```

### Example

```js
  require('ng-filesort')(files)
```
